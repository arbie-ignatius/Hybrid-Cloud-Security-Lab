# Redaction Policy

This repository contains sanitized documentation and configurations from a personal hybrid homelab project.

## Redacted Items
- Public IP addresses
- Home network identifiers and SSIDs
- Hostnames tied to real identities
- API keys, tokens, passwords, private keys
- Serial numbers and MAC addresses (when applicable)

## Allowed Documentation
- Architecture diagrams
- VLAN IDs and private IP addressing (RFC1918)
- Screenshots with sensitive details blurred
- Configuration examples using placeholders such as <REDACTED> or <TOKEN>

All configurations are intentionally sanitized to prevent disclosure of sensitive information while preserving architectural integrity.
